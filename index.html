電費計算器

使用說明：

1. 此計算機適用於合租族群

2. 每間房間需有配置個人電表

3. 計算機會自動計算出公共用電以及個人用電

4. 最後輸出的結果即為每位室友應繳金額

5. 輸入時請確保"本期"以及"上期"位置輸入正確

生活範例：

假設我們有兩個室友，他們分別為A和B。他們住在一起，共同分攤電費。以下是每個人的上期和本期電表數值，以及整個住處的本期總用電度數和總電費：

  • 本期總用電度數：800 度
  • 本期電費總金額：2400 元
  • A 的上期電表數值：100
  • A 的本期電表數值：300
  • B 的上期電表數值：150
  • B 的本期電表數值：450

計算步驟：

  1. 計算每個人的用電量：
    • A 的用電量 = 300 - 100 = 200 度
    • B 的用電量 = 450 - 150 = 300 度
  2. 計算個人總用電量：
    • 個人總用電量 = 200 + 300 = 500 度
  3. 計算公共用電量：
    • 公共用電量 = 總用電度數 - 個人總用電量 = 800 - 500 = 300 度
    • 每人分攤的公共用電量 = 300 / 2 = 150 度
  4. 計算每度電費：
    • 每度電費 = 總電費 / 總用電度數 = 2400 / 800 = 3 元/度
  5. 計算每個人的電費：
    • A 的電費 = (個人用電量 + 公共用電量) * 每度電費 = (200 + 150) * 3 = 350 * 3 = 1050 元
    • B 的電費 = (個人用電量 + 公共用電量) * 每度電費 = (300 + 150) * 3 = 450 * 3 = 1350 元
